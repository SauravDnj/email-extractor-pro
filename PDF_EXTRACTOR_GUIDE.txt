## ğŸ› Troubleshooting

### Problem: "No emails found in PDF"
**Solutions:**
1. PDF might be scanned image (not text-based)
2. Use OCR software first to convert
3. Check if PDF has selectable text

### Problem: "All emails already selected"
**Solutions:**
1. Upload a different PDF
2. Reset history to start fresh
3. Check available count in statistics

### Problem: "Excel file not downloading"
**Solutions:**
1. Check browser download settings
2. Disable popup blockers
3. Try different browser
4. Check disk space

### Problem: "PDF upload fails"
**Solutions:**
1. Check file size (max 50MB)
2. Ensure it's a valid PDF
3. Try different browser
4. Check file isn't corrupted
